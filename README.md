# Green Basket Grocery Shop

A React grocery shop project with:

- Home page
- Product listing with search and category filter
- Cart page
- About and contact pages
- WhatsApp order button
- Admin login
- Admin dashboard
- Add, edit, and delete product pages

## Run

```bash
npm install
npm start
```
